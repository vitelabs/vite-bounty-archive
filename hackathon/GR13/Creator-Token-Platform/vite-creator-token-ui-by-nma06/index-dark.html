Welcome to VITE Creator Token Plateform

This platform allows VITE members to have their own tradeable token (creator token) via a bonding curve.

A bonding curve is an application that mints tokens through buy and sell functions. The price increases as more tokens are in circulation. The amount paid for the token is deposited in a common account.

At any point in time, a token can be withdrawn from the active supply, and a proportional part of the communal deposit can be taken with.
