== Description ==

** NOT FOR USE IN PRODUCTION CODE...YET **
Vite Payments for Woocommerce is blockchain technology powering your Woocommerce store, providing payment functionality in your store utilizing the Vite protocol.

== Installation ==

1. Upload the plugin zip to Wordpress site
2. Activate in the admin plugin dashboard
3. Configure and enable through Woocommerce 'Payment' settings
